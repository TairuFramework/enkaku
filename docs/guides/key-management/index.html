Key management

Creating keys and signers​

The @enkaku/token package provides APIs to create private keys and signers, notably:

  • randomPrivateKey() to create a random private key of 32 bytes.
  • getTokenSigner() to create TokenSigner object from a private key.
  • randomTokenSigner() to create an OwnTokenSigner object containing the private key.

Encoding and decoding keys​

The @enkaku/token package provides the encodePrivateKey() function to encode a private key as a base64 string, and the decodePrivateKey() function to decode it back.

Securely storing keys​

The @enkaku/protocol package exports the KeyEntry and KeyStore types implemented by the following key management packages:

  • @enkaku/browser-keystore for Web browsers environments.
  • @enkaku/expo-keystore for Expo (React Native) environments.
  • @enkaku/node-keystore for Node environments.
