Core concepts

Messages​

Enkaku messages are JSON objects using the JSON Web Token (IETF RFC 7519) format.

Messages may be signed and verified using the JSON Web Signature (IETF RFC 7515) standard.

JSON Web Token​

Enkaku uses JSON Web Token (JWT) to represent messages.

Validation​

Messages supported by a given protocol are defined using JSON Schemas and validated by the server logic.
